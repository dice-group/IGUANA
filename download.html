Release Files

The release files zip contains the iguana jar file, a start script and an example benchmark suite yaml file. To adjust the configuration to your needs and create the benchmark you want have a look at our Wiki to create your configuration at.

Wiki.

The same example benchmark suite configuration can also be found at

Example Suite. Download

To execute your benchmark use either the start script

~$ ./start-iguana.sh iguana.yml &

or

~$ java -jar iguana.corecontroller-x.y.z.jar iguana.yml
